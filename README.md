## simple-java-api

### Requirements

- Java 21 Temurin
- Apache Maven 3.6.3 or higher
- PostgreSQL 9.6 or higher
- Docker 20.10.21 or higher
- Docker Compose 2.13.0 or higher

### Getting started

Clone the repository:

```bash
git clone git@github.com:timtwiest/simple-java-api.git
```

Build the project:

```bash
mvn clean install
```
